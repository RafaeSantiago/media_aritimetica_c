\# Cálculo de Média Aritmética em C



Este projeto consiste em um programa desenvolvido em linguagem C que calcula a média aritmética de quatro provas de um aluno e determina sua situação acadêmica com base na média obtida.



\## Funcionalidades

\- Leitura das notas de quatro avaliações

\- Cálculo da média aritmética

\- Verificação da situação do aluno:

  - Média maior ou igual a 6: \*\*Aprovado\*\*

  - Média menor que 6: \*\*Recuperação\*\*

\- Exibição do resultado no console



\## Tecnologias utilizadas

\- Linguagem C

\- Compilador GCC

\- Execução via terminal (PowerShell / CMD)



\## Lógica do programa

O programa solicita ao usuário a entrada de quatro notas, realiza o cálculo da média aritmética e utiliza uma estrutura condicional para verificar se o aluno foi aprovado ou encaminhado para recuperação, conforme o valor da média calculada.



\## Como compilar

Utilize o compilador GCC no terminal: gcc media\_aritimetica.c -o media\_aritimetica

\## Como executar

./media\_aritimetica
